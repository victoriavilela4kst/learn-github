🌟 Curso Git 4 Women - Atividade Prática 🌟

Bem-vinda ao meu espaço de aprendizado e prática! 🎉 Neste repositório, estou consolidando os conhecimentos adquiridos na aula, explorando os seguintes tópicos essenciais:

✨ Tópicos Praticados

- Criação de Repositório: Início de tudo!
- Gestão de Issues: Organizando tarefas e prioridades.
- Discussions: Lugar para debates e troca de ideias.
- Pull Requests: Colaboração e revisão de código.
- Criação e Gerenciamento de Wikis: Documentação organizada e acessível.

🎯 Objetivo

Fortalecer o domínio sobre Git e GitHub através da aplicação prática, e aprimorar as habilidades de trabalho em projetos colaborativos. 💪✨

💻 O Que Está Incluído Neste Repositório?

Este repositório contém exemplos e experimentos realizados ao longo da atividade, incluindo:

- Issues criadas para organizar os desafios e tarefas.
- Discussões iniciadas para simular ambientes colaborativos.
- Pull requests realizados para validar alterações no código.
- Documentação adicional feita nas Wikis.

🛠️ Ferramentas Utilizadas
- GitHub
- Criatividade e foco! 🌟
